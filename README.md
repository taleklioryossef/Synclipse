# Synclipse - LinkedIn Auto-Refresher

A professional Chrome extension for automatically refreshing LinkedIn pages.

## Installation
1. Download this repository as ZIP
2. Extract the files
3. Go to chrome://extensions/
4. Enable "Developer mode"
5. Click "Load unpacked" and select the extracted folder

## Usage
1. Navigate to any LinkedIn page
2. Click the Synclipse icon
3. Set refresh interval (1-60 seconds)
4. Set duration (1 minute - 24 hours)
5. Click "Start Refreshing"
